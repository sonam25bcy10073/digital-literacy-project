## 📌 Project Overview
This project is part of the Digital Literacy course. The aim of this project is to develop essential digital skills such as creating a professional online presence, understanding safe internet practices, and learning effective digital communication.

---

## 📂 Project Structure

- task-1-presentation/ → Infographic on digital literacy  
- task-2-portfolio/ → Screenshots of digital profiles  
- task-3-platforms/ → Coding platform and Google Form  
- task-4-email-etiquette/ → Emails and checklist  
- task-5-cybercrime/ → Case study and prevention tips  
- report/ → Final project report  

---

## 🧩 Task Summary

### Task 1: Digital Literacy Infographic
In this task, I created an infographic using Canva to explain digital literacy concepts such as safe internet usage and digital tools.

### Task 2: Student Digital Portfolio
I created and updated my profiles on GitHub and LinkedIn to build my professional digital presence.

### Task 3: Coding and Collaboration Platforms
I solved a coding problem on HackerRank and created a quiz using Google Forms.

### Task 4: Email Etiquette and Social Media Guidelines
I wrote professional emails and created a checklist for responsible social media use.

### Task 5: Cybercrime Awareness
I prepared a case study on phishing and created a prevention checklist to promote online safety.
